# Travel Stories: A Visual Portfolio of My Trips

## Project Type
Coding Project

## Overview
Travel Stories is a personal travel website that showcases my recent trips through curated itineraries, photos, and visual storytelling.

Each trip has its own page with:
- a trip overview
- day-by-day itinerary sections
- photo galleries
- trip highlights
- interactive place lists with additional details

## Target Audience
This website is designed for:
- friends and classmates
- people interested in travel inspiration
- people interested in photography
- potential collaborators who want to view my work

## Project Goals and Deliverables
- Build a responsive homepage introducing my travel portfolio
- Create individual trip pages with itinerary sections and media galleries
- Implement a consistent navigation system, including a mobile-friendly hamburger menu
- Add interactive features using JavaScript
- Apply accessibility best practices such as semantic HTML, alt text, color contrast, and keyboard navigation

## Current Features
- Homepage with featured trips
- Fukuoka trip page
- Las Vegas trip page
- Dropdown navigation for trips
- Interactive image lightbox
- Interactive list of featured places on trip pages
- Accessible image alt text and keyboard-friendly interactions
